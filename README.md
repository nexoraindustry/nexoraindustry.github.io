## Setup Instructions

1. Download all files to your computer
2. Place all HTML files in the root directory
3. Place CSS file in `assets/css/` directory
4. Place JavaScript file in `assets/js/` directory
5. Open `index.html` in a browser to view the website

## Customization

1. **Company Information**: Update contact details in all HTML files
2. **Images**: Replace Unsplash image URLs with your own images
3. **Colors**: Modify CSS variables in `style.css` for color changes
4. **Content**: Update text content in HTML files as needed

## Deployment to GitHub Pages

1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to repository Settings > Pages
4. Select "main" branch as source
5. Your site will be published at `https://username.github.io/repository-name`
